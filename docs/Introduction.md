# Introduction

Welcome to the Linguist documentation.

Linguist is a privacy-first browser translation extension for Chrome, Firefox, and other compatible browsers. It helps you translate web pages, selected text, subtitles, messages, and custom text input directly in your browser without switching between websites or applications.

Unlike many translation extensions, Linguist is not tied to a single translation provider. You can choose from multiple built-in translation services, use [offline translation](./guides/OfflineTranslation.md) on your device, [connect your own APIs](./CustomModules.md), or even integrate custom translators and local machine translation systems.

## What can Linguist do?

* Translate entire web pages
* Translate selected text on any website
* Translate text entered manually
* Use offline translation for improved privacy
* Listen to text with text-to-speech
* Save translations to your history and personal dictionary
* Configure automatic translation rules
* Integrate custom translation services and APIs
* Create your own translators and speech modules using custom modules

## Privacy first

Linguist is designed with privacy in mind. When using the built-in offline translator, translations are processed directly on your device without sending text to external services. Linguist is free, open source, and does not collect or sell user data.

## Flexible by design

Whether you prefer Google Translate, Microsoft Translator, DeepL, a self-hosted translation server, an LLM, or a completely custom solution, Linguist gives you control over how translations are performed. Advanced users can extend Linguist with custom modules and integrate virtually any translation backend.
